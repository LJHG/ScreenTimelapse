# ScreenTimelapse
ScreenTimelapse is a tool  written in python to record time-lapse video for your screen.

## requirements
1. PIL
2. cv2
3. numpy

## how to run
python screenTimelapse.py -wi 1920 -he 1080 -o demo.avi -r 100 -w 1
